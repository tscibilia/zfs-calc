# ZFS Calculator

An interactive web-based calculator for ZFS storage configurations, inspired by the Synology RAID Calculator.

## About

This tool helps users plan and calculate ZFS storage pool configurations, including:

- Usable capacity calculations
- RAIDZ configurations (RAIDZ1, RAIDZ2, RAIDZ3)
- Mirror and stripe setups
- Storage efficiency estimates
- Redundancy planning

## Features

- Interactive disk configuration
- Real-time capacity calculations
- Visual representation of storage layouts
- Support for various VDEV configurations
- Disk failure tolerance information
